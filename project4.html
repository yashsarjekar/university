ECOMMERCE SHOPPING MOBILE APP


THIS APP IS HYBRID APP
WHICH WORKS ON BOTH PLATFORM ANDROID AND IOS

Description

Here in this project I have build an shopping ecommerce app where a saler can sell products and customers can buy this product from any remote location.Technologies used in this project is Flutter framework and dart programming language which is developed and maintain by Google. The advantage of using this framework is that by using same code it can build a app for ANDROID as well as IOS. So We can say it is an hybrid app which works on both the platforms i.e Android as well as IOS. The code editor used in this project is Vs code.
